# 在React.js安裝bootstrap和node-sass
![](1.png)
安裝bootstrap需要typescript、jquery、popper.js

均在project內安裝，非全域安裝
jquery與popper.js指定版本
``` shell
$ npm install --save typescript
$ npm install --save jquery@1.9.1
$ npm install --save popper.js@1.14.7 
```
