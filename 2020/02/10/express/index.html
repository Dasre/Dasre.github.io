express router上個人常用的指令

req -> request

res -> response

  • req.params
    • 取得路徑參數值
    • ex
  • res.json()
    • 輸出json資料
  • res.render()
    • 渲染指定畫面
  • res.redirect()
    • 網址重新導向

其餘大致上是邏輯的處理。

補充

  • req.params. …
    • 撈出路由設定的資料
  • req.query. …
    • 取得網址的參數
  • app.use()
    • 使用middleware
    • 類似一層一層的過濾，中間可以處理http的request、response或一些檢查動作等等。
